# 4.Métodos de interpolación
#### Se usan para estimar valores desconocidos entre datos conocidos. Se trazan puntos y se calcula un valor intermedio.


## Codigo 1:

```python
x1 = 1
y1 = 2
x2 = 3
y2 = 6

y = 4
print(y)
```
## Resultado 1:
#### 4
