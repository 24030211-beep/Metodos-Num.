# 3.Método de aproximaciones sucesivas
#### También llamado punto fijo. Consiste en usar un valor inicial y reemplazarlo repetidamente en una fórmula hasta acercarse a la solución.


## Codigo 1:

```python
x = 1
x = (x + 2) / 2
print(x)
```
## Resultado 1:
#### 1.5


## Codigo 2:

```python
x = 2
x = (x + 5) / 2
print(x)
```
## Resultado 2:
#### 3.5


## Codigo 3:

```python
x = 4
x = (x + 1) / 2
print(x)
```
## Resultado 3:
####  2.5

```python
x = 10
```
