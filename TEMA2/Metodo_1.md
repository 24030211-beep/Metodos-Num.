# 1.Métodos de intervalo
#### Son métodos numéricos que buscan raíces dentro de un intervalo donde la función cambia de signo. Se parte de dos valores iniciales y se reduce poco a poco el rango hasta encontrar una aproximación.


## Codigo 1:
```python
a = 1
b = 2

print("Intervalo inicial:", a, b)

```
## Resultado 1
#### Intervalo inicial: 1 2


## Codigo 2:
```python

a = 3
b = 5

print("Intervalo:", a, b)
```
## Resultado 2
#### Intervalo: 3 5



## Codigo 3:
```python
a = -1
b = 1

print("Intervalo:", a, b)

```
## Resultado 3
#### Intervalo: -1 1



## Codigo 4:
```python
a = 0
b = 10

print("Intervalo:", a, b)

```
## Resultado 4
#### Intervalo: 0 10



## Codigo 5:
```python
a = 2
b = 4

print("Intervalo:", a, b)

```
## Resultado 5
#### Intervalo: 2 4
