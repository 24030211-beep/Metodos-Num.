# 2.Método de bisección
#### Consiste en dividir repetidamente un intervalo en dos partes y elegir la mitad donde se encuentra la raíz. Es un método seguro y sencillo.


## Codigo 1:

```python
a = 1
b = 2
m = (a + b) / 2

print(m)
```
## Resultado 1:
#### 1.5


## Codigo 2:

```python
a = 3
b = 5
m = (a + b) / 2

```
## Resultado 2:
#### 4.0
