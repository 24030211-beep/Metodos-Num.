# 6. Coversión Estrecha
#### Sucede cuando se convierte un dato de un tipo más amplio a uno más limitado, por ejemplo de float a int. Al hacerlo, se eliminan decimales o parte de la información original.


## Codigo 1:

```python
x = int(3.9)
print(x)
```


## Codigo 2:

```python
y = int(8.99)
print(y)
```


## Codigo 3:

```python
a = float(7)
print(a)
```


## Codigo 4:

```python
n = int(-2.7)
print(n)
```


## Codigo 5:

```python
precio = int(19.95)
print(precio)
```
